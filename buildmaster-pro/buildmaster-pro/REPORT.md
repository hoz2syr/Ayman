## 6. نظام Responsive الحالي

### 6.1 فئاتBreakpoints

```css
/* Mobile First Approach */
xs: < 640px      /* Extra Small - phones */
sm: 640-767px   /* Small - large phones */
md: 768-1023px  /* Medium - tablets */
lg: 1024-1279px /* Large - laptops */
xl: ≥ 1280px    /* Extra Large - desktops */
```

### 6.2 Classes المساعدة

```css
/* إظهار/إخفاء */
mobile-only, tablet-only, desktop-only
xs-hidden, sm-hidden, md-hidden, lg-hidden, xl-hidden

/* المسافات */
xs-p-2, xs-p-3, xs-p-4, sm-p-3, sm-p-4, md-p-4, md-p-5

/* حجم النص */
xs-text-xs, xs-text-sm, sm-text-sm
```
